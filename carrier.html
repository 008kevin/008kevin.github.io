NeoTelecom Help

/carrier

/carrier create name

Creates a carrier with the given name. All settings will be the default, meaning everything is free. Make sure to change this with /carrier set (see below)

/carrier remove name

Removes the carrier with this name.

/carrier set name option value

Change a setting of a carrier, like the prices.

/carrier list page

Shows a list of carriers, with their important info. Page can be specified to go to a specific page.

/carrier info carrier

Displays information about a carrier.

/carrier peer carrier add peer

Send a peering request to a carrier.

/carrier peer carrier remove peer

Remove a peer from your carrier.

/carrier peer carrier accept peer

Accepts a peering request from a carrier.
